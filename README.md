# 高等教育个性化学习资源智能体系统

## 项目简介

基于大模型技术体系，融合前沿AI技术，构建高等教育个性化学习资源体系。该系统通过多智能体协同工作，为学生提供个性化、多模态的学习资源生成和管理服务。

## 核心功能

### 1. 对话式学习画像自主构建
- ✅ 自然语言对话方式构建学生画像
- ✅ 不少于8个维度的动态画像
- ✅ 支持画像随学随新、持续更新
- 维度：知识基础、认知风格、学习偏好、易错点偏好、学习节奏、目标导向、学科兴趣、学习习惯

### 2. 多智能体协同的资源生成
- ✅ 多智能体架构设计（11个专业智能体）
- ✅ 至少6种类型个性化资源生成
- 资源类型：课程讲解文档、思维导图、练习题、拓展阅读、教学视频脚本、代码实操案例

### 3. 个性化学习路径规划
- ✅ 基于画像的动态路径规划
- ✅ 多类型资源精准推送
- ✅ 分阶段、分步骤的学习计划

### 4. 智能辅导答疑（加分项）
- ✅ 即时多模态答疑服务
- ✅ 文字解答、图解说明、分步讲解
- ✅ 常见错误提醒、知识点拓展

### 5. 学习效果评估（加分项）
- ✅ 多维度精准评估（6个评估维度）
- ✅ 动态调整推送策略
- ✅ 学习方案持续优化

## 快速开始

### 环境要求
- Python 3.9+
- OpenAI API Key

### 安装与运行

1. **克隆/下载项目**
```bash
cd 软件杯
```

2. **配置API Key**
编辑 `.env` 文件，填入你的 OpenAI API Key：
```
OPENAI_API_KEY=sk-your-api-key-here
```

3. **安装依赖**
```bash
pip install -r requirements.txt
```

4. **启动服务**
```bash
python main.py
```
或双击 `start.bat` (Windows)

5. **访问系统**
打开浏览器访问：https://localhost:8000

## 使用指南

### 第一步：注册学生信息
在左侧边栏填写学号、姓名、专业等信息，点击"注册/登录"

### 第二步：构建学习画像
在对话区选择"画像构建"模式，与AI助手自然对话：
- 介绍你的专业背景
- 描述你的学习目标和兴趣
- 分享你的学习习惯和困难

### 第三步：生成学习资源
在对话中输入类似：
- "生成 数据结构 二叉树 的学习资料"
- 或点击"一键生成全部资源"

### 第四步：查看学习路径
系统会自动规划学习路径，在"学习路径"标签查看

### 第五步：学习与评估
- 浏览和阅读各类学习资源
- 遇到问题随时在对话区提问
- 定期进行学习效果评估

## 智能体说明

| 智能体 | 角色 | 功能 |
|--------|------|------|
| ProfileAgent | 学习画像分析师 | 通过对话构建多维度学习画像 |
| ContentAgent | 课程内容生成专家 | 生成个性化课程讲解文档 |
| ExerciseAgent | 习题生成专家 | 生成多类型练习题 |
| MindMapAgent | 知识结构梳理专家 | 生成知识思维导图 |
| ReadingAgent | 拓展阅读推荐专家 | 生成拓展阅读材料 |
| VideoAgent | 多媒体教学设计专家 | 生成教学视频脚本 |
| CodeAgent | 编程实践指导专家 | 生成代码实操案例 |
| PathAgent | 学习路径规划师 | 规划个性化学习路径 |
| TutorAgent | 智能辅导教师 | 提供即时答疑服务 |
| EvaluationAgent | 学习评估分析师 | 多维度学习效果评估 |
| OrchestratorAgent | 多智能体协调器 | 协调多智能体协同工作 |

## 技术亮点

1. **LangGraph多智能体架构**：基于LangGraph构建智能体工作流，实现智能体间的协调与协作
2. **多模态资源生成**：支持文档、思维导图、视频脚本、代码等多种格式
3. **动态画像更新**：通过持续对话不断完善学生画像，实现"随学随新"
4. **个性化学习路径**：基于画像和资源智能规划学习步骤
5. **即时智能辅导**：多角度、多形式的答疑解惑
6. **多维效果评估**：实时跟踪学习数据，精准评估学习效果
